# Methodology

## 1. Threat Modeling
Based on ransomware kill chain:
- Initial access  
- Privilege escalation  
- Lateral movement  
- Data exfiltration  
- Encryption  

## 2. Detection Engineering
Use Sigma, YARA, and KQL to detect:
- Precursors  
- Execution  
- Encryption behavior  
- Exfiltration  

## 3. Response
SOC playbooks for:
- Precursor detection  
- Active ransomware  
- Identity compromise  

## 4. Recovery
Backup strategy, isolation, and rebuild.

## 5. Compliance
NYCRR Part 500  
ISO 27001
